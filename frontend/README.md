# Sree Sakthi Paper Tech

**Sree Sakthi Paper Tech** - Your trusted partner for buying and selling second-hand paper machines, spare parts, and equipment in Coimbatore, India.

## Project Info

This project is a React application built with Vite, TypeScript, and Tailwind CSS.

## Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository:**
    ```sh
    git clone <YOUR_GIT_URL>
    cd <YOUR_PROJECT_NAME>
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

To build the project for production:

```sh
npm run build
```
